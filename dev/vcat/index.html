Concatenation

To vertically concatenated dataframes use 

vfld_df = read_v(vfldfile, select=[:ID,:TT])
vobs_df = read_v(vobsfile, select=[:ID,:TT])

df = vcat(vfld_df,vobs_df,cols=:union)
